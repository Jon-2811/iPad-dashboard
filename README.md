# iPad Desk Dashboard v7

## v7の変更点

- カード透明度の下限をさらに下げ、水のように薄いLiquid Glass風に調整
- Settingsの「カード透明度」は 0.02 まで下げられます
- 通知音量の上限を2倍に拡張
- デフォルト通知音量も上げました
- ポモドーロ完了回数に応じて魚が増える「育成ウィジェット」を追加
- Settingsページに簡易付箋メモを追加
- v6までの機能は継続
  - Googleカレンダー同期
  - 天気アイコン
  - 傘判断
  - 背景画像複数設定
  - 自由入力タイマー
  - ページループ

## 透明度調整

Settings → カード透明度  
背景を主役にしたい場合は 0.02〜0.08 がおすすめです。
文字が読みにくい場合は 0.12〜0.20 くらいに上げてください。

## 通知音量

Settings → 通知音量  
最大値をv6の2倍にしました。iPad本体の音量も上げてください。
