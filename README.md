# iPad Desk Dashboard v5

## v5の変更点

- カードの黒さを弱め、背景画像が見えやすいデザインに変更
- 背景画像の初期表示を少し濃く変更
- 通常タイマーに自由入力を追加
  - 分・秒を好きに入力して設定可能
- 国試カウントダウン下の「正式日程が出たら...」文言を削除
- ページ切替をループ化
  - Home → Timer → Settings → Home
  - Settings → Home もスワイプで移動可能
- タイマー終了時の音通知はv4から継続
- 天気アイコン、傘判断、背景画像複数設定も継続

## iPadで音が鳴らないとき

iPad/Safariは自動再生制限があるため、最初に一度「開始」または「通知音テスト」を押してください。
その後はタイマー終了時に音が鳴りやすくなります。

## 背景画像

Settings > 背景画像を選ぶ から複数枚選択できます。
画像はiPad内に保存され、GitHubにはアップロードされません。

## Googleカレンダー同期

`google-apps-script-calendar.gs` をGoogle Apps Scriptに貼り付けてWebアプリ化し、SettingsにURLを貼ってください。
