# iPad Desk Dashboard v6

## v6の変更点

- カードをiPhoneのLiquid Glass風にさらに透明化
- Settingsに「カード透明度」スライダーを追加
- Settingsに「通知音量」スライダーを追加
- 通知音をより目立つチャイムに変更
- 背景画像がより見えやすいように背景オーバーレイを薄く調整
- v5の機能は継続
  - 自由入力タイマー
  - ページループ
  - 傘判断
  - 背景画像複数設定
  - Googleカレンダー同期

## 音が小さい場合

Settings → 通知音量 を上げてください。
iPad/Safariは自動再生制限があるため、最初に一度「通知音テスト」または「開始」を押してください。

## カードがまだ濃い/薄い場合

Settings → カード透明度 で調整してください。
背景画像を主役にしたいなら 0.10〜0.18 くらいがおすすめです。
