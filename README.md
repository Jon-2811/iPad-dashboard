# iPad Desk Dashboard v8

## v8の変更点

- カード透明度をさらに下げ、最薄時は水のようにかなり透明なLiquid Glass風に変更
- 育成ウィジェットを削除
  - タイマー操作の邪魔になっていたため
- 付箋メモを移動
  - Home: 今日の予定の右横
  - Timer: ポモドーロ/通常タイマー画面内
- ポモドーロの下バーは従来通り、作業セッション完了ごとに進みます
- 通常タイマーの自由設定を使いやすい位置に戻しました
- 通知音量2倍設定は継続

## 透明度

Settings → カード透明度  
最小値は 0.005 です。
文字が読みにくい場合は 0.04〜0.10 程度まで上げてください。

## 付箋

HomeとTimerのどちらからでも同じ付箋を編集できます。
付箋をタップすると削除されます。
